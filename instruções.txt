1. Abra o projeto na sua máquina

2. entre na pasta Back-End

3. rode "npm install" no terminal para instalar as dependencias

4. crie o arquivo .env com a senha do seu banco de dados (DATABASE_URL="postgresql://postgres:senha@localhost:porta/nome_do_banco")

5. rode "npx prisma migrate dev" no terminal para copiar o banco pronto que está no schema.prima para o banco da sua máquina

6. rode "npx prisma generate" no terminal para ler o banco de dados do schema

6. rode "npm start" no terminal para inicar o projeto
